# Common-validation

common-validations is a library to provide common validations, some exception and messages for restful web-service

Validation is defined in api.json file, and the model generated by swagger will contain validation constraint annotation. 

## Documentation

Defined validator in model schema description and/or property description (in api.json). Use '|' as separator

```$xslt
Survey": {
	"type": "object",
	"properties": {
	  "id": {
		"type": "string",
		"description": "Unique identity UUID | @ValidUUID"
	  },
	  "name": {
		"type": "string",
		"description": "Name of supplier | @NotBlank | @MaxLength(100)"
	  },
	  "questions": {
		  "type": "array",
		  "description": "List question of survey | @NotEmptyArray | @MaxItems(5)",
		  "items": {
			"$ref": "#/components/schemas/Question"
		  }
	  },
	  ...
```
